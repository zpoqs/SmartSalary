# 薪智汇控 - 智能薪酬管理系统 (SmartSalary)

## 📌 项目定位
本项目为**软件工程专业课程设计团队作业**。项目旨在通过现代软件工程的设计模式，构建一个高可靠性、智能化、可视化的企业级薪金与考勤管理平台。

---

## 🚀 核心系统特性与设计亮点

### 1. 📂 数据库优先架构与双引擎降级灾备 (SQLite & MySQL)
* **标准数据库驱动**：系统彻底摆脱传统文件读写方式，全面改用标准 SQL 驱动，极大提升了多用户并发下的数据一致性与持久化性能。
* **双数据库无缝降级**：系统默认支持网络级 **MySQL**。当服务器连接失败或不可达时，系统将**自动降级至本地 SQLite 数据库**进行业务容灾备份，保证系统具备 100% 的高可用性。

### 2. 🤖 AI 智能决策顾问 (内置通义千问 Qwen3.7-Max)
* **深度对话与分析**：对接阿里云 DashScope API，启用智能思考模型（Reasoning Engine），为企业管理者提供实时的薪酬政策解读、个人薪资变化分析等服务。
* **隐私保密数据交互**：采用**高层数据抽象交互**机制。发送给 AI 的数据仅包含脱敏后的抽象统计数据（如部门均值、极值分布、异动人数），杜绝任何员工个人敏感信息（PII）泄露。
* **离线自动降级**：当 API 服务不可用或未配置 Key 时，AI 助手将自动切换至本地规则模板引擎，继续提供业务帮助。

### 3. 📊 多彩数据可视化引擎 (QtCharts 深度优化)
* **定制多彩主题**：舍弃了传统的单色调图表，重新设计了高对比度、视觉舒适的多色调配色方案。
* **多维数据洞察**：提供部门薪资分布饼图、薪酬极值柱状图以及员工年龄结构趋势图，帮助管理层一目了然地把控公司的人才结构与财务成本。

### 4. ⚙️ 薪资自动化清算与微调机制
* **自动化计算流**：一键导入员工考勤数据后，系统会自动根据考勤扣减项、五险一金政策、个人所得税率等进行精准的实发工资清算。
* **双表格精细化管理**：
  * **历史只读账单**：清晰呈现上一月份的已发放工资明细，供合规审计。
  * **当月草稿账单**：提供直观的网格编辑界面，支持管理员双击实时微调基本工资和岗位工资，并即时完成整表重算。

## 🛠️ 运行环境与依赖要求
* **开发框架**：Qt 5.14.1 (或兼容版本)
* **编译器**：MinGW 64-bit / MSVC
* **数据库**：MySQL 5.7+ / 本地 SQLite 3
* **API 服务**：阿里云百炼平台（DashScope API）

---

## ⚙️ 配置文件 `config.ini` 说明
项目启动时会读取根目录下的 `config.ini` 文件，您需要根据实际环境配置以下参数：

```ini
[Database]
; 数据库驱动类型：QMYSQL 或 QSQLITE
Driver=QSQLITE
HostName=localhost
Port=3306
DatabaseName=salary_system
UserName=root
Password=root
; 本地 SQLite 数据库文件名
SqliteDbName=salary_system.db

[AI]
; 阿里云百炼 API Key
ApiKey=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
; 模型名称，默认推荐 qwen3.7-max
ModelName=qwen3.7-max
; 是否启用深度思考
EnableThinking=true
```

## 💻 编译与构建步骤
1. 打开命令行，进入项目根目录：
   ```bash
   cd SmartSalary
   ```
2. 执行 qmake 生成 Makefile：
   ```bash
   qmake SmartSalary.pro
   ```
3. 执行编译（Windows 环境下使用 mingw32-make）：
   ```bash
   mingw32-make
   ```
4. 运行生成的可执行程序：
   ```bash
   ./release/SmartSalary.exe
   ```
